# agendamento
A simple booking application Java+Wicket+Hibernate+Facebook API

Uma simples aplicação para agendamentos Java+Wicket+Hibernate+Facebook API

Features

- Booking details page to save new booking requests;
- Booking list page to show saved booking requests. You can search for date, accept or reject requests.
- Mailing. Send e-mails on accepting or rejecting requests;
- Booking list page has browser notification feature. Showing notification on new booking request.
- Booking details page has Facebook integration. It gets user name from facebook account when logged-in on facebook.
- Login capabilities implemented using Java Filters. Only list page requires login.

Funcionalidades

- Pagina de detalhes de agendamentos para salvar agendamentos;
- Pagina de listagem de agendamentos para mostrar os agendamentos feitos. Pode-se buscar por data, aceitar ou rejeitar agendamentos.
- E-mail. Envia e-mails quando solicitações são aceitas ou rejeitadas.
- Pagina de listagem possui funcionalidade de notificação em navegadores. Sempre que uma nova solicitação é criada, uma notificação é exibida.
- Pagina de detalhes tem integração com o Facebook. Caso o usuario esteja logado no facebook a pagina ja recupera algumas informações como o nome da pessoa.
- Sistema de login utilizando Java Filters. Apenas a pagina de listagem exige login.
